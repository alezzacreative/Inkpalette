Inspired by [Appalette](https://github.com/Carlosarturo28/Appalette) plugin for Sketch.

## Install
- Copy `inkpalette.py`, `inkpalette_db.py`, `inkpalette.inx` in `~/.config/inkscape/extensions`
- Restart Inkscape

## Run
Extensions -> Color -> Inkpalette

- Linux
- Windows
